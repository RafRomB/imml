API Reference
============

This is the main reference for imlearn. For a further guide on effectively incorporating these functions into
your workflows, please see the user guide.

List of modules
---------------

:mod:`imvc.ampute`: Transformers for missing view amputation.

:mod:`imvc.cluster`: Unsupervised clustering algorithms.

:mod:`imvc.datasets`: Utilities to load popular datasets.

:mod:`imvc.decomposition`: Matrix decomposition algorithms.

:mod:`imvc.impute`: Transformers for missing view imputation.

:mod:`imvc.preprocessing`: Methods for preprocessing multi-view datasets.

:mod:`imvc.utils`: Various utilities to help with development.
